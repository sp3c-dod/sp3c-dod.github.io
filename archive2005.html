  Stop Talking Got Frag Ads!

Posted: Mon. April 4, 2005 9:44 a.m. EST
Annoyed by talking ads on Got Frag? Go to your C:\Windows\system32\drivers\etc\ folder (winnt if running Windows 2000) and open the file "hosts" in notepad. After the line "127.0.0.1       localhost" hit enter TWICE and paste the following two lines:
127.0.0.1 k84105.gotfrag.com
127.0.0.1 v841n5.gotfrag.com
127.0.0.1 m8154n.gotfrag.com
127.0.0.1 5jjkl1.gotfrag.com
127.0.0.1 j4185x.gotfrag.com
127.0.0.1 l81941.gotfrag.com
127.0.0.1 m8154n.gotfrag.com
127.0.0.1 n75915.gotfrag.com
127.0.0.1 k417cz.gotfrag.com
127.0.0.1 m4715n.gotfrag.com
If you notice some talking ads that this does not block simply go to "View Source" in your browser and do a serach for "displayadvertisment", you will be able to find the server that the advertisment is on that way. I have done this for talking AIM ads as well. Find a samples hosts file by clicking here.

Note: If you have known extensions hiding enabled in Windows this file might get saved with .txt on the end of it. It will NOT work with .txt on the end of it. It must be just "host". Also, you must close and reopen MSIE for the settings to take effect.
