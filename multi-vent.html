How to use multiple versions of Ventrilo at once

The first step is to download the versions you don't have:
For 2.3.0+ visit ventrilo.com.
For 2.2.0 click here.
For 2.1.4 click here. (works on 2.1.3 servers)

  • If you don't have any version installed then install one version of Ventrilo.
  • Once installed (or if already installed) locate Ventrilo.exe.
    By default this file is located at C:\Program Files\Ventrilo\
  • Rename this file something different such as Ventrilo2.1.4.exe
  • Uninstall the current version and install a different version.
  • Repeat for each version you wish to install.
